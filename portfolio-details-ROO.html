Detalles del portafolio ERP

Información de proyecto

  • Categoria: Implementación ERP
  • Cliente: Representaciónes Ortopédicas de Occidente
  • Fecha del proyecto: 14 Diciembre , 2022
  • Ir a la página: Virtual Private Server

Implementacion del Sistema para la administración de negocio.

Se llevó a cabo la colección de la información del negocio y se diseñaron los procesos de negocios basicos para la comercializacion de articulos de ortopedia. Este proyecto consistió en crear un sitio en la nube para instalar una aplicación ERP - Enterprise Resource Planning para la operacion de la empresa Representaciones Ortopédicas de Occidente. El sistema contiene un mecanismo de respaldo automático para la aplicación y los datos.
