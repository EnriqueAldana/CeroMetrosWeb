# CeroMetrosWeb
## Descripción

Sitio web, incluye:
- Inicio
- Acerca de
- Servicios
- Portafolio
- Equipo
- Contacto


Platformas soportadas
-------------------

- macOS 10.12+
- Linux
- Windows 10

Requirementos del sitio
-------------------

- Cualquier navegador con soporte de javascript, html
- Cualquier servidor web que incluya PHP 8 en adelante.
